## How to develop?

1. You need `pipenv`

```bash
pip install pipenv
```

1. Install all requirements [for development]

```bash
pipenv install --dev
```

1. Run application!

```bash
python -m src.main
``` TODO: may raise ModuleNotFoundError exceptions depending on project structure
